About MarketPlace

MarketPlace is a clean Django marketplace template. It includes categories, listings with photos, user reviews, and email-based authentication via django-allauth. The UI is responsive and uses unified cu-card styles.

  • Listings: create, edit, search, filters
  • Images gallery for ads
  • User profiles, ratings, comments
  • Ready to deploy (Render/Heroku notes in README)
